FromHex(string)

Creates a color from a hex triplet. A hex triplet is either three or six hexadecimal digits that represents an RGB Color.

An example of a hex triplet is "09C" or "0099CC", which both represents the same color.

FromString(string)

Creates a color from a string. Currently, only numeric RGB values are supported. This method works the same was as FromHex(string) except that is required as hash sign before the hex value.

An example of a numeric RGB value is "#09C" or "#0099CC", which both represents the same color.
